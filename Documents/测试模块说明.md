# TireflySquadNavigation 测试模块说明

## 模块概述

`TireflySquadNavigationTest` 是插件的 **Editor-only** 测试 / 演示模块（`Type = EditorNoCommandlet`），不会打入 Shipping 包。

包含内容：
- 测试用 C++ 类（AI 棋子、AIController、靶标、批量生成器）
- 测试用 BT 节点（目标选择、模拟攻击）
- 自动化测试（2 个单元测试 + 3 个功能测试）

## 测试用 C++ 类

| 类名 | 父类 | 说明 |
|------|------|------|
| `ATsnTestChessPiece` | `ACharacter` + `ITsnTacticalUnit` | AI 棋子，实现完整接口，挂载全部插件组件 |
| `ATsnTestAIController` | `AAIController` | 集成 `UCrowdFollowingComponent`，自动关闭 RVO |
| `ATsnTestTargetDummy` | `ACharacter` | 靶标，含 `TsnEngagementSlotComponent` + 可选 Spline 巡逻 |
| `ATsnTestSpawner` | `AActor` | 批量生成器，支持 Line / Column / Grid / Circle 四种阵型 |
| `ETsnTestFormation` | `enum class` | 阵型枚举 |

## 测试用 BT 节点

| 节点 | 类型 | 说明 |
|------|------|------|
| `UTsnTestBTTask_SelectTarget` | 同步 Task | 搜索最近敌方目标，写入 Blackboard |
| `UTsnTestBTTask_Attack` | 异步 Task | 模拟攻击倒计时 |

## 自动化测试

在 UE 编辑器中通过 **Session Frontend → Automation** 运行。

### 单元测试

| 测试 | 注册路径 | 说明 |
|------|---------|------|
| `TsnSlotCalculationTest` | `TireflySquadNavigation.Test.Unit.SlotCalculation` | 验证槽位坐标计算公式（7 个用例） |
| `TsnRepulsionVectorTest` | `TireflySquadNavigation.Test.Unit.RepulsionVector` | 验证排斥力双阶段衰减公式（8 个用例） |

### 功能测试

需要预先创建 `MAP_TsnAutoTest` 地图（见下方前置条件）。

| 测试 | 注册路径 | 说明 |
|------|---------|------|
| `SlotAssignmentTest` | `TireflySquadNavigation.Test.Functional.SlotAssignment` | 直接调用 RequestSlot API 验证多环分配 |
| `StanceAvoidanceTest` | `TireflySquadNavigation.Test.Functional.StanceAvoidance` | MoveToLocation 验证双层防线避障 |
| `FullBattleCycleTest` | `TireflySquadNavigation.Test.Functional.FullBattleCycle` | BT 驱动完整战斗循环 |

### 前置条件

**MAP_TsnAutoTest**：功能测试需要一个极简地图：
1. 创建新 Level → 保存为 `/TireflySquadNavigation/Content/Test/Maps/MAP_TsnAutoTest`
2. 放置 Plane（Scale 60×60）作为地面
3. 放置 NavMeshBoundsVolume（Extent 3000×3000×500）
4. Build Navigation → 保存

**BT/BB 资产**（仅 FullBattleCycleTest 需要）：
- `BB_TsnTest`：两个键 —— `TargetActor`(Object, Actor) + `HasTarget`(Bool)
- `BT_TsnTestChessPiece`：完整行为树（详见 design.md）

## 演示地图

为避免同图多场景互相选错目标，演示内容拆分为 3 张独立地图，均需手动搭建（详见 design.md §演示地图详细设计）：
- `MAP_TsnDemo_Siege`：6 单位围攻单靶标（多环槽位验证）
- `MAP_TsnDemo_Skirmish`：红蓝对战（完整战斗循环 + 目标切换）
- `MAP_TsnDemo_MovingTarget`：移动目标追击（槽位追踪与重算）

## 运行测试

1. 打开 UE 编辑器
2. Window → Developer Tools → Session Frontend → Automation
3. 展开 `TireflySquadNavigation` → `Test`
4. 勾选测试 → Start Tests
5. 观察结果（绿色 ✓ / 红色 ✗）
